Users & Notes API: Learn, Test & Explore

Project Overview

This is a RESTful API built with Node.js, Express, and PostgreSQL using Sequelize as the ORM. The API allows you to perform CRUD operations on users and their notes, with a one-to-many relationship between users and notes.

Technology Stack
  • Backend: Node.js with Express
  • Database: PostgreSQL
  • ORM: Sequelize
Database Schema

The database uses the following schema:

// User Model { firstName: STRING, lastName: STRING, email: STRING (unique) } // Note Model { content: TEXT, userId: INTEGER (foreign key) }
API Endpoints

Users

  • GET /users - Get all users with their notes
  • GET /users/:id - Get a specific user with their notes
  • POST /users - Create a new user
  • PUT /users/:id - Update a user
  • DELETE /users/:id - Delete a user

Notes

  • GET /notes - Get all notes with their users
  • GET /notes/:id - Get a specific note with its user
  • POST /notes - Create a new note
  • PUT /notes/:id - Update a note
  • DELETE /notes/:id - Delete a note

GET Get All Users

Retrieves all users with their notes

curl http://localhost:3000/users

GET Get Single User

Retrieves a specific user by ID with their notes

curl http://localhost:3000/users/{id}

POST Create New User

Creates a new user

curl -X POST http://localhost:3000/users -H "Content-Type: application/json" -d '{"firstName": "John", "lastName": "Doe", "email": "john@example.com"}'

PUT Update User

Updates an existing user by ID

curl -X PUT http://localhost:3000/users/{id} -H "Content-Type: application/json" -d '{"firstName": "John", "lastName": "Doe", "email": "john@example.com"}'

DELETE Delete User

Deletes a user by ID

curl -X DELETE http://localhost:3000/users/{id}
